# Project & Automation Ideas

A running list of things we could build as a group or automate for managing The Owlery and the Rest of the Owl Discord.

## Discord Community Tools

- **Progress milestone bot** — tracks member achievements like first PR, first code review, first snippet shared
- **Weekly digest bot** — summarizes active threads and shared links from the past week into a single Discord post
- **Role assignment bot** — assigns roles based on skill level, activity, or self-reported progress
- **Code review request bot** — post a snippet, request reviewers, track who's responded
- **Random pair-matching bot** — weekly matchups for code buddy sessions or casual rubber ducking

## Learning & Leveling Up

- **Shared interview prep repo** — a group-curated collection of common interview questions and answers
- **TIL tracker** — members submit "today I learned" entries, aggregated into a browsable archive
- **Roadmap generator** — takes a skill gap as input and outputs a suggested learning path

## Organization Management

- **Owlery repo organizer bot** — auto-labels and categorizes repos by language or topic as they're added
- **Member onboarding automation** — checklist or script for adding new members to both GitHub and Discord
- **Owlery → Discord digest** — pipes new repos and snippets posted to the org into a Discord channel

## Miscellaneous

- **Shared dotfiles repo** — a collectively maintained dev environment setup the group can contribute to and fork from
- **Polyglot code challenge repo** — someone posts a prompt, members submit solutions in different languages
